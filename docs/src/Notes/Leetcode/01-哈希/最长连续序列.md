解题思路可以看答题情况里面的，这里再记录一下一些收获：

可以利用哈希将查找的时间复杂度从$O(n)$ 降至$O(1)$  

泛型中不可以用基本数据类型，比如：
```java
Set<Integer> set = new HashSet<Integer>();
//而不是
Set<int>
```

**什么时候使用哈希法**，当我们需要查询一个元素是否出现过，或者一个元素是否在集合里的时候，就要第一时间想到哈希法。
