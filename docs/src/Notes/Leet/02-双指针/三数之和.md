关键点在于去重，但也要考虑到{-1, -1, 2} 和 {0, 0, 0,x,x,x,x}的情况
对于前者，我们用
```java
if(i>0 &&nums[i]==nums[i-1]) continue;
```

对于后者，我们用:注意，0，0，0可以通过，但是一旦后面有其他的数，那么就没法通过了
首先找到三元组，然后对元素进行去重；而不是一进入右边大于左边的情况（也就是循环开始时），就去重，这样会使得提前退出循环
